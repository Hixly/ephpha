# Ephpha

> An AI email assistant that helps you write better emails, craft subject lines that get opened, and know exactly when to hit send.

**Live:** [ephpha.ai](https://ephpha.ai)

Most email tools save you keystrokes. Ephpha saves you opens. It scores your subject line, suggests rewrites for the body, and tells you the smart moment to send based on the recipient and context — so your emails don't get buried.

## What it does

- **Subject line scoring** — opens-likelihood feedback in real time
- **AI-assisted rewrites** — tone, brevity, hook
- **Smart send timing** — recommended send window per recipient
- **Drafting from prompts** — describe the email, get a draft

## Built with

- Next.js
- TypeScript
- Anthropic Claude
- Deployed on Vercel
